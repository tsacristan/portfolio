# Portfolio Thurian Sacristan

Portfolio personnel moderne avec Vue 3, Vite et TypeScript.

## 🚀 Démarrage rapide

### Installation

```bash
npm install
```

### Lancer en développement

```bash
npm run dev
```

Le site sera accessible à `http://localhost:5173`

### Build de production

```bash
npm run build
```

Les fichiers compilés seront dans le dossier `dist/`

## 📋 Stack technique

- **Frontend**: Vue 3 Composition API
- **Build**: Vite 5.4.21
- **Typage**: TypeScript
- **Styling**: CSS Scoped (SCSS-like)
- **Icons**: Emojis natifs

## 📁 Structure

```
src/
  ├── components/     # Composants Vue
  ├── utils/          # Données et utilitaires
  ├── types/          # Types TypeScript
  ├── composables/    # Logique réutilisable
  ├── App.vue         # Composant racine
  └── main.ts         # Point d'entrée
```

## 🎨 Sections du site

- **Hero**: Présentation principale
- **Projets**: Portfolio de projets (interface bureau interactive)
- **Compétences**: Référentiel des compétences BUT
- **Passions**: Strava + Powerlifting
- **Contact**: Formulaire de contact + réseaux sociaux
